## Getting Started

```
- cd backend
- py -3.11 -m venv venv
- ./venv/Scripts/activate
- pip install -r requirements.txt
```

```
- In the root (or simultaneously use : cd..)
- npm install
- npm install -D concurrently
```

``` npm run dev 
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
